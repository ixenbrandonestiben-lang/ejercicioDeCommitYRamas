# 📦 +++ inventario_tienda +++

## ------Nombre del Proyecto------

Sistema de Inventario para Gestión de Productos

---

## ------ Descripción del Sistema ------

Este proyecto consiste en un **sistema de inventario desarrollado en Python** que permite gestionar diferentes productos dentro de una lista.

El sistema funciona desde la terminal y permite realizar operaciones básicas para la administración de productos como agregar, listar, actualizar, eliminar, calcular el valor total de los productos y ver la cantidad actual de los productos.

Este programa fue creado con el objetivo de poner en practica los conocimientos adquiridos en clases para practicar conceptos de programación como:

* listas
* diccionarios
* funciones
* manejo de errores
* estructuras de control

---

## ▶️ Instrucciones de Uso

1. Clonar el repositorio:

```
git clone https://github.com/ixenbrandonestiben-lang/ejercicioDeCommitYRamas.git
```

2. Entrar a la carpeta del proyecto:

```
cd ejercicioDeCommitYRamas
```

3. Ejecutar el programa:

```
python main.py
```

4. Seleccionar una opción del menú:

* Agregar producto
* Listar productos
* Actualizar producto
* Eliminar producto
* Calcular valor total del inventario
* productos con poca cantidad.
* Salir del sistema
